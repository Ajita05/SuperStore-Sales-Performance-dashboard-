Superstore Sales Dashboard | Tableau

An interactive Tableau dashboard built using the Sample Superstore dataset to analyze sales performance, profit, customer segments, and regional trends. The dashboard helps uncover business insights through interactive visualizations and filters.

Live Dashboard

🔗 Tableau Public: https://public.tableau.com/views/TBIProject/KPICARDDashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

Project Objectives

- Analyze sales and profit performance
- Compare sales across regions and states
- Identify top-performing product categories
- Evaluate customer segment contributions
- Provide interactive filtering for business analysis

Tools Used

- Tableau Desktop
- Microsoft Excel
- Data Visualization
- Business Intelligence

Dashboard Highlights

- KPI Cards
- Sales Trend Analysis
- Regional Performance
- Category & Sub-category Analysis
- Customer Segment Analysis
- Interactive Filters

Key Business Insights

- Technology generated the highest sales.
- Some product sub-categories had high sales but low profitability.
- The West region outperformed other regions in total sales.
- Consumer customers contributed the largest share of sales.
